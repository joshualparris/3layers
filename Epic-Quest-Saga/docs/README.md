# Epic Quest Saga Documentation Suite

This `docs` suite is the operating system for building, running, and upgrading `Epic-Quest-Saga` as a professional indie game/app product.

It is grounded in:

- `Epic-Quest-Saga` app source (especially `artifacts/continuity-app` and `replit.md`)
- `__Alfie Dnd` canonical materials, including:
  - `docs/repo/repository-inventory.md`
  - `docs/character/alfie-current-inventory.md`
  - `Alphie_Complete_Inventory.md`
  - `docs/Checkpoints/SKT_Dubbo_Resume_Checkpoint_4_Handover.md`
  - `ClaudeHandover27042026.md`
  - `ChatGPThandover27042026.md`

## How to Use

1. Start each planning cycle with `03-production/roadmap-and-milestones.md`.
2. Convert priorities into execution tickets from `03-production/master-backlog.md`.
3. Keep lore and gameplay aligned using `01-product/narrative-canon-bible.md`.
4. Validate all major changes against `04-quality/qa-test-strategy.md` and `04-quality/risk-register.md`.
5. When new source material is added in `__Alfie Dnd`, update `00-source-integration/source-ingestion-map.md` first.
