# Roadmap And Milestones

## Delivery Themes

- T1: Canon reliability and source ingestion.
- T2: Session-time usability and tactical support.
- T3: Engineering resilience and release discipline.

## Milestone M1 - Documentation Foundation (Now)

Goals:

- Establish complete professional docs suite.
- Map sources from `__Alfie Dnd` into product governance.

Exit criteria:

- Docs structure committed and reviewed.
- Traceability map and source index complete.

## Milestone M2 - Canon And Import Engine

Goals:

- Implement schema versioning and migration hooks.
- Build markdown import adapters for checkpoint and handover docs.

Exit criteria:

- Repeatable import for at least 3 high-value upstream docs.
- Conflict detection marks unresolved mismatches.

## Milestone M3 - Session Command Center

Goals:

- Add "session readiness" dashboard.
- Add unresolved conflict and obligation panels.

Exit criteria:

- One-click session prep flow.
- Sub-10-second retrieval for key mission and character facts.

## Milestone M4 - Quality And Operations Hardening

Goals:

- Add robust regression checks.
- Introduce backup/export and recovery tooling.

Exit criteria:

- Stable release checklist execution.
- Proven restore workflow from snapshot.

## Milestone M5 - Product Expansion

Goals:

- Optional multi-campaign support.
- Optional collaboration and sync architecture planning.

Exit criteria:

- Campaign abstraction validated.
- Technical spike docs for backend-ready evolution.
